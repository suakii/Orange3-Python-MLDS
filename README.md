이 책은 데이터 과학 및 기계학습의 다양한 적용 분야를 다루며, 특히 오렌지3와 파이썬을 활용하여 문제를 해결하는 프로젝트 활동들로 구성되어 있습니다. 각 챕터에서는 실제 데이터에 기반한 프로젝트를 통해 이론을 실제로 구현하고, 머신러닝의 개념과 응용 방법을 익힐 수 있게 구성되었습니다.
